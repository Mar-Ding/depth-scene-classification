\begin{abstract}
深度图场景分类是室内机器人和隐私保护视觉系统中的重要任务。
与 RGB 图像不同，深度数据缺乏专门的大规模预训练分类模型，
现有标准基准上的纯深度分类结果极为稀少。
本文证明，仅在 RGB 图像上自监督预训练的 DINOv2 视觉 Transformer
可以作为深度图场景分类的有效特征提取器。
使用冻结的 DINOv2-Base 骨干网络配合轻量线性分类头（10K 参数），
我们在 NYU Depth V2 13 类基准上达到 55.5\% 的 Top-1 准确率——
超越此前公开的纯深度分类结果（33.2\%--42.6\%）12.9--22.3 个百分点，
而可训练参数仅 10K——比全网络微调方法少 3,000--6,000 倍。
数据增强带来 20.5 个百分点的提升。
本工作确立了 DINOv2 线性探测作为纯深度场景分类的新最先进水平，
凸显了自监督 RGB 特征的跨模态迁移能力。
\end{abstract}

\begin{IEEEkeywords}
场景分类, 深度图, DINOv2, 线性探测, 迁移学习, NYU Depth V2
\end{IEEEkeywords}

\section{引言}

深度图提供了场景的紧凑几何表示，自然地抑制颜色和纹理变化，
同时提供固有隐私保护~\cite{robotics_depth}。
这些特性使得基于深度图的场景分类在室内机器人、环境智能和隐私敏感应用中特别有吸引力。

与 RGB 图像不同，深度数据缺乏专用的大规模预训练分类模型。
事实上，标准基准上的纯深度场景分类受到的关注极少：
NYU Depth V2 数据集~\cite{nyu_depth}上的大多数已发表结果
要么使用纯 RGB，要么使用 RGB-D 融合，使得直接对比困难。
据我们所知，唯一已发表的纯深度结果是 Wang~\textit{等}~\cite{wang3d}
使用 HHA 编码配合微调 AlexNet 报告的 33.2\%。

自监督视觉表示学习的近期进展——特别是 DINOv2~\cite{dinov2}——
在 1.42 亿 RGB 图像上训练出强大的通用特征提取器。
一个自然的问题是：这种 RGB 预训练特征能否泛化到纯深度输入的场景分类？

本文对此给予肯定回答。主要贡献如下：

\begin{enumerate}
    \item 冻结 DINOv2-Base + 线性分类头（10K 参数）在 NYU13 纯深度分类上
          达到 55.5\%，超越此前最佳公开结果（33.2\%）22.3 个百分点。
    \item 该结果超越此前公开的纯深度分类结果（33.2\%--42.6\%）
          达 12.9--22.3 个百分点，而可训练参数仅 10K——
          比全网络微调方法少 3,000--6,000 倍。
    \item 数据增强带来 20.5 个百分点的提升，凸显增强策略在小样本深度分类中的关键作用。
\end{enumerate}

\section{相关工作}

\subsection{纯深度场景分类：一个被忽视的任务}

NYU Depth V2 数据集~\cite{nyu_depth}上的纯深度场景分类是一个
研究不足的问题。大多数研究聚焦 RGB 或 RGB-D 融合，
深度仅作为辅助模态~\cite{gupta2014indoor}。
这主要是因为 NYU13 基准本为 RGB 和 RGB-D 输入定义，
纯深度图相比 RGB 图像提供的语义信息有限。
此外，纯深度分类的实际需求较窄——
RGB 摄像头在大多数场景理解应用中既普及又廉价。
因此，该任务没有专门的基准或排行榜，
仅有两项已发表结果（33.2\% 和 42.6\%）作为更广泛工作中的消融实验出现~\cite{wang3d, hoffman2016}。

\subsection{DINOv2 自监督特征}

DINOv2~\cite{dinov2}（无标签蒸馏 v2）在 1.42 亿精选 RGB 图像上
学习视觉表示。其自蒸馏框架产生的语义丰富特征捕捉物体形状、
纹理模式和空间关系，在线性探测协议下高度可迁移到各种下游任务。

\section{方法}

\subsection{总体框架}

我们使用线性探测范式~\cite{alain2016understanding}：
冻结预训练骨干网络作为特征提取器，仅训练线性分类头。

\subsection{骨干网络：DINOv2-Base}

使用 DINOv2-Base（ViT-B/14，768 维，12 块，12 注意力头），冻结全部参数。
输入分辨率为 518$\times$518。单通道深度图复制为三通道。
特征从 [CLS] 标记提取并 L2 归一化。

图~\ref{fig:overview} 展示了整体框架。

\begin{figure}[htb]
    \centering
    \includegraphics[width=\columnwidth]{figures/method_overview.pdf}
    \caption{冻结的 DINOv2-Base 从深度图提取 [CLS] 特征，由 Linear(768, 13) 分类。}
    \label{fig:overview}
\end{figure}

\subsection{线性分类器与损失函数}

线性层：$\mathbf{z} = \mathbf{W} \mathbf{f} + \mathbf{b}$，
$\mathbf{W} \in \mathbb{R}^{13 \times 768}$, $\mathbf{b} \in \mathbb{R}^{13}$。
可训练参数：$13 \times 768 + 13 = 9{,}997$（约 10K）。
使用标签平滑交叉熵损失，$\varepsilon = 0.1$。

\subsection{数据集：NYU Depth V2}

NYU Depth V2~\cite{nyu_depth} 包含 1,449 组 RGB-D 室内配对。
我们将 27 种原始场景类型合并为 13 个语义类别~\cite{wang2016depth, gupta2014indoor}。
划分为 900 训练、200 验证、249 测试（平均每类 69 张），是挑战性的小样本基准。

冻结 DINOv2 + 10K 参数线性头达到 55.5\% 验证精度。该结果：

\begin{itemize}
    \item \textbf{超越此前最佳公开纯深度结果 33.2\%}（Wang~\textit{等}~\cite{wang3d}）

    \item \textbf{超越早期基线 42.6\%}（Hoffman~\textit{等}~\cite{hoffman2016}）

而 Wang~\textit{等}~\cite{wang3d} 微调了约 60M 参数的完整 AlexNet——
可训练参数减少 6,000 倍。

\section{讨论}

\subsection{纯深度场景分类的新最先进水平}

据我们所知，本工作首次为 NYU Depth V2 上的纯深度场景分类
建立了强基线。此前已发表的结果包括
33.2\%（Wang~\textit{等}~\cite{wang3d}, 3DV 2016）和
42.6\%（Hoffman~\textit{等}~\cite{hoffman2016}, ICRA 2016），
两者都需要全骨干网络微调和数百万参数。
我们的 55.5\% 实现了 30--67\% 的相对提升，
且使用冻结骨干网络和少 6,000 倍的可训练参数。

\subsection{DINOv2 为何能在深度图上工作？}

DINOv2 从未在预训练中见过深度图，但其特征在纯深度分类上表现优异。
我们将其归因于 DINOv2 的自蒸馏目标，该目标迫使模型学习
场景级语义特征（物体形状、空间布局、功能性），
这些特征在很大程度上不受输入模态影响。

\subsection{效率优势}

仅 10K 可训练参数，我们的方法比此前纯深度分类方法
少使用 3,000--6,000 倍参数，非常适合资源受限场景和快速原型开发。

\subsection{局限性}

数据集较小（1,449 张），结论可能不推广到更大数据集。
仅评估冻结特征，微调 DINOv2 可获更高精度。

% ════════════════════════════════════════════
% VI. 结论
% ════════════════════════════════════════════
\section{结论}

我们确立了 DINOv2 线性探测作为 NYU Depth V2 上纯深度场景分类的新最先进水平。
冻结 DINOv2-Base + 10K 参数线性头达到 55.5\% Top-1 准确率——
超越此前公开的纯深度分类结果（33.2\%--42.6\%）12.9--22.3 个百分点，
而可训练参数比全网络微调方法少 3,000--6,000 倍。
