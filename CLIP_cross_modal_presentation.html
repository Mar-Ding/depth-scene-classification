Emergence

视觉模型的涌现

01

LLM 路径

文本生成预训练 → 涌现推理、翻译、代码能力 → 指令微调对齐。预训练损失函数单一（next token prediction），但产生了泛化智能。

02

视觉模型路径

图像生成 / 对比预训练 → 涌现视觉理解能力 → 轻量适配到具体任务。CLIP、扩散模型都展现出超越原始训练目标的泛化能力。

03

共同假设

预训练视觉模型（生成式或对比式）都内化了通用视觉知识，只需轻量适配就能泛化到未见任务和新模态。

01 · RESEARCH BACKGROUND
05 / 25
Google 2026 · arXiv:2604.20329

Vision Banana

Image Generators are Generalist Vision Learners

核心

扩散模型预训练 → 指令微调 → 所有视觉任务的输出表示为 RGB 图像。语义分割 = 生成分割掩码图，深度估计 = 生成深度图。

SOTA

在分割、深度估计等任务上超越 SAM 3 和 Depth Anything 系列专精模型。证明生成式预训练本身就是通用视觉学习器。

启示

我们的方法与之互补：CLIP（对比式）做分类，而非生成。共同验证了"预训练视觉模型 = 通用特征提取器"。

Problem Statement

跨模态传感器场景理解

借用CLIP

27类室内

NYU Depth V2 数据集，27 种室内场景类型全覆盖

方法

对比学习

NT-Xent Loss，深度-RGB 正样本对拉近，负样本推远

评估

Zero-shot

训练时不需文本标签，推理时直接用类别名做零样本分类

愿景

通用框架

任意传感器（深度/雷达/声纳）只需一个轻量 Adapter 复用 CLIP

Vision Banana

生成式

  • 扩散模型基座
  • 输出 = RGB 图像
  • 不侧重跨模态
  • 模型大（完整扩散）
  • 任务不限（分割/深度/边缘/检测）
Our Method

对比式

  • CLIP 图文对比基座
  • 输出 = 文本类别匹配
  • 专攻跨模态
  • 极轻量（528K 参数）
  • 专注分类/匹配
04 · ANALYSIS
18 / 25
Strengths

方法优势

极轻量

528K 参数完成跨模态对齐。完整训练仅 ~7 分钟（RTX 4060）。可部署在边缘设备。

零样本

训练时不需要文本标签，只需深度-RGB 对。推理时直接用类别名做 zero-shot 分类。

易迁移

换传感器只需重新训练 Adapter，CLIP 完全不动。深度/雷达/声纳数据都适用。

04 · ANALYSIS
19 / 25
Limitations

局限性

精度

信息天花板

纯深度不含颜色和纹理，场景区分能力天然有限。63% 相对 RGB 性能在预期内。

过拟合

样本不足

600 训练样本对 528K 参数 Adapter 偏少。Val Acc 在 7-15% 间随机震荡。

分布

长尾问题

Top-4 类占总数 65%。尾部类别（< 5 样本）几乎无法学习，拉低整体精度。

Key Takeaways

核心结论

✓

跨模态适配可行

冻住 CLIP + 528K 参数 Adapter 即可让深度图实现 zero-shot 场景分类，Top-1 19.5%。

✓

通用视觉学习者

与 Vision Banana 一致，预训练视觉模型可作为通用特征提取器，轻量适配即可泛化到新任务。

⚠

精度受限于模态

纯深度约达 RGB 上限的 63%。对结构相似场景区分困难，需融合多模态信息。
